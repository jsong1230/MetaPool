# MetaPool — PRD

## 1. 프로젝트 개요
- **목적**: Metadium 블록체인 위에서 META 토큰을 사용한 Binary(Yes/No) 예측 마켓 플랫폼. 간단한 Yes/No 구조로 진입 장벽을 낮추고, 온체인 투명성으로 신뢰를 확보한다.
- **대상 사용자**:
  - META 보유 암호화폐 투자자 (시장 방향 예측으로 추가 수익)
  - 스포츠/이벤트 결과 예측을 즐기는 사용자
  - MetaLotto 기존 유저 (지갑+META 보유, 온보딩 마찰 0)
- **핵심 가치**:
  - 극저 가스비 (Metadium) + 저렴한 최소 참여금 (100 META ~$0.11)
  - 블록체인 투명성: 모든 베팅/결과가 온체인 기록
  - 간단한 UX: Binary(Yes/No) 구조, 복잡한 파생상품 없음

## 3. 비기능 요구사항
- **성능**: 가스비 최소화 (베팅 ~80,000 Gas, 클레임 ~60,000 Gas)
- **보안**: ReentrancyGuard, Checks-Effects-Interactions 패턴, 오버플로우 방지(Solidity 0.8+), maxBet 제한
- **확장성**: 단일 MetaPool.sol 컨트랙트로 시작, v2에서 모듈화 가능

## 4. 기술 스택
- Smart Contract: Solidity ^0.8.19, OpenZeppelin v5.x, Hardhat
- Frontend: React, ethers.js v6
- Blockchain: Metadium (Mainnet Chain ID: 11, Testnet Chain ID: 12)
- 지갑: MetaMask (window.ethereum)
- DB: 없음 (온체인 전용)

## 5. 범위 외 (Out of Scope)
- P2 기능: 자동 클레임 알림(R-6), 뱃지 시스템(U-4)
- v2 기능: 사용자 마켓 생성(DAO), 다중 선택지 마켓, 오라클 자동화, 다중서명 관리자
- 모바일 앱 (웹 반응형만 지원)
- 백엔드 서버/DB (순수 온체인 + 프론트엔드)
