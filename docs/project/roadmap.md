# 프로젝트 로드맵

## Milestone 0: 프로젝트 기반 구축 ✅
- **목표**: 개발 환경 설정, 시스템 설계 문서 작성
- **작업**: Hardhat 프로젝트 초기화, React 프로젝트 스캐폴딩, 시스템 설계(/init-dev + /design)
- **완료 기준**: Hardhat 컴파일/테스트 환경 동작, React 개발 서버 기동, 설계 문서 완료

## Milestone 1: 스마트 컨트랙트 핵심 ✅
- **목표**: 예측 마켓의 핵심 라이프사이클(생성 -> 베팅 -> 결과 확정 -> 클레임/환불)을 스마트 컨트랙트로 구현
- **기능**: F-01, F-02, F-03, F-04, F-05, F-06, F-07, F-08
- **결과**: 113개 테스트 PASS

## Milestone 2: 스마트 컨트랙트 확장 ✅
- **목표**: 이의제기 시스템, 긴급 중단, 설정 관리 등 운영/거버넌스 기능 추가
- **기능**: F-09, F-10, F-11, F-12
- **결과**: 191개 테스트 PASS

## Milestone 3: 프론트엔드 핵심 ✅
- **목표**: 지갑 연결부터 마켓 탐색, 베팅, 결과 확인, 클레임까지 핵심 사용자 플로우 구현
- **기능**: F-13, F-14, F-15, F-16, F-17, F-18, F-19, F-20, F-21, F-22, F-23
- **결과**: 전체 프론트엔드 빌드 성공, 로컬 Hardhat 노드 연동 확인

## Milestone 4: 프론트엔드 확장 ✅
- **목표**: 프로필/통계, 다국어, 검색/정렬 등 사용자 경험 확장 기능 구현
- **기능**: F-24, F-25, F-26, F-27, F-28, F-29, F-30, F-31, F-32
- **결과**: i18n 4개 언어, 리더보드, 풀 차트, 검색/정렬 구현

## Milestone 5: 통합 테스트 및 배포 ✅
- **목표**: 전체 기능 통합 테스트, 배포 스크립트
- **작업**: E2E 테스트 5개 시나리오, 보안 체크리스트 13항목, 배포 스크립트
- **결과**: 225개 테스트 PASS, 로컬 Hardhat 배포 + 샘플 마켓 생성 검증 완료

## 추가 구현 ✅
