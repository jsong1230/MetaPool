1. MetaPool이란?

MetaPool은 Metadium 블록체인 위에서 운영되는 탈중앙화 예측 마켓 플랫폼입니다. 암호화폐 가격, 스포츠 경기, 날씨, 정치 등 다양한 주제에 대해 Yes 또는 No로 예측하고, 정확한 예측을 한 참여자가 보상을 받는 구조입니다.

모든 베팅과 보상은 스마트 컨트랙트를 통해 투명하게 처리되며, META 토큰(Metadium의 네이티브 토큰)을 사용합니다.

핵심 특징: Binary(Yes/No) 구조로 간단한 참여 / 블록체인 기반 완전 투명 / 낮은 가스비 / 최소 100 META부터 참여 가능

2. 시작하기 — 지갑 설정

MetaPool을 이용하려면 MetaMask 지갑에 Metadium 네트워크를 추가해야 합니다.

MetaMask 네트워크 설정

Network NameMetadium Mainnet
RPC URLhttps://api.metadium.com/prod
Chain ID11
SymbolMETA
Block Explorerhttps://explorer.metadium.com
MetaMask 설치Chrome Web Store에서 MetaMask 확장 프로그램을 설치하고 지갑을 생성합니다.
네트워크 추가MetaMask → 설정 → 네트워크 → 네트워크 추가 → 위의 정보를 입력합니다.
네트워크 전환네트워크 선택 드롭다운에서 "Metadium Mainnet"을 선택합니다.

구매 후 출금 절차

4. 마켓 탐색 & 베팅

마켓 탐색

메인 화면에서 활성 마켓 목록을 볼 수 있습니다. 카테고리 필터(Crypto, Sports, Weather 등)를 사용하여 관심 있는 주제를 빠르게 찾으세요. 각 마켓 카드에는 질문, 현재 배당률, 풀 규모, 마감 시간이 표시됩니다.

베팅 방법

방향 선택마켓 카드에서 "Yes 베팅" 또는 "No 베팅" 버튼을 클릭합니다.
금액 입력슬라이더 또는 직접 입력, 퀵 버튼(100/500/1K/5K/10K META)으로 베팅 금액을 설정합니다.
정보 확인현재 배당률과 예상 수익을 확인합니다. 배당률은 실시간으로 변동합니다.
트랜잭션 서명"베팅 확인" → MetaMask에서 트랜잭션을 서명합니다. 가스비는 극소량입니다.
참고: 하나의 마켓에서는 한 방향(Yes 또는 No)만 선택할 수 있습니다. 같은 방향으로 추가 베팅은 가능하지만, 반대 방향으로는 베팅할 수 없습니다.

5. 배당률과 보상 구조

배당률 계산

배당률은 Yes 풀과 No 풀의 비율로 결정됩니다. 적은 사람이 선택한 방향일수록 배당률이 높습니다.

Yes 40%
No 60%
Yes 풀40,000 META
No 풀60,000 META
총 풀100,000 META
Yes 배당률100,000 / 40,000 = 2.50x
No 배당률100,000 / 60,000 = 1.67x

보상 계산 예시

결과가 Yes로 확정된 경우:

Yes 풀 (승리)10,000 META
No 풀 (패배)15,000 META
플랫폼 수수료 (2%)15,000 × 2% = 300 META
분배 가능 금액14,700 META
A (6,000 META 베팅)6,000 + 8,820 = 14,820 META
B (4,000 META 베팅)4,000 + 5,880 = 9,880 META

6. 결과 확정 & 클레임

결과 확정

이벤트가 종료되면 관리자가 공식 데이터 소스를 기반으로 결과(Yes/No/Void)를 확정합니다. 결과는 블록체인에 영구 기록됩니다.

보상 클레임

"내 베팅" 탭 확인결과가 확정된 마켓에 "적중" 표시가 됩니다.
"보상 클레임" 클릭클레임 버튼을 눌러 보상을 수령합니다.
MetaMask 서명트랜잭션을 서명하면 보상이 지갑으로 입금됩니다.
무효(Void) 마켓: 이벤트 취소 등으로 마켓이 무효 처리되면, 모든 참여자에게 베팅 원금이 전액 환불됩니다.

8. 자주 묻는 질문

MetaPool은 안전한가요?
모든 베팅과 보상은 스마트 컨트랙트로 처리되며, 블록체인에 투명하게 기록됩니다. 관리자도 사용자의 자금을 임의로 인출할 수 없습니다.
최소/최대 베팅 금액은?
최소 100 META (~$0.11), 최대 100,000 META (~$110)입니다.
양쪽 모두에 베팅할 수 있나요?
아니요. 하나의 마켓에서는 Yes 또는 No 중 한 방향만 선택 가능합니다. 같은 방향 추가 베팅은 허용됩니다.
베팅을 취소할 수 있나요?
블록체인에 기록된 트랜잭션은 취소할 수 없습니다. 베팅 전 신중하게 결정하세요.
수수료 구조는?
베팅 시 수수료는 없습니다 (가스비만 부담). 정산 시 패배 풀의 2%만 플랫폼 수수료로 차감됩니다.
결과에 동의하지 않으면?
1,000 META를 스테이킹하고 이의를 제출할 수 있습니다. 이의가 인정되면 스테이킹 반환, 기각 시 몰수됩니다.
보상을 클레임하지 않으면?
보상은 스마트 컨트랙트에 안전하게 보관됩니다. 기한 제한 없이 언제든 클레임 가능합니다.
MetaLotto와 같이 쓸 수 있나요?
네! 같은 MetaMask 지갑, 같은 Metadium 네트워크, 같은 META 토큰을 사용합니다. 별도의 추가 설정 없이 바로 이용 가능합니다.
