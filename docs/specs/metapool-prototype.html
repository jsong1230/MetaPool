배당률과 보상 구조

각 마켓에는 Yes 풀과 No 풀이 있습니다. 많은 사람이 선택한 방향은 배당률이 낮아지고, 적은 사람이 선택한 방향은 배당률이 높아집니다.

예시: Yes 풀 40,000 META / No 풀 60,000 META
→ Yes 배당률 2.5x / No 배당률 1.67x
→ Yes에 1,000 META 베팅 시 예상 수익: 2,500 META (순이익 1,500)

플랫폼은 패배 풀의 2%만 수수료로 차감합니다. 나머지 98%는 전액 승리 측에 분배됩니다.

마켓 카테고리

MetaPool은 6가지 카테고리의 예측 마켓을 지원합니다: 암호화폐 가격 예측, 스포츠 경기 결과, 날씨 예보, 정치 이벤트, 엔터테인먼트, 그리고 기타 모든 주제. 각 마켓의 결과는 공식 데이터 소스를 기준으로 확정됩니다.

무효 마켓 (Void)

이벤트가 취소되거나 결과 판정이 불가능한 경우, 마켓은 Void(무효) 처리되며 모든 참여자에게 베팅 원금이 전액 환불됩니다.

자주 묻는 질문

최소/최대 베팅 금액은?

최소 100 META, 최대 100,000 META입니다.

양쪽 모두에 베팅할 수 있나요?

아니요. 하나의 마켓에서는 한 방향(Yes 또는 No)에만 베팅할 수 있습니다. 같은 방향으로 추가 베팅은 가능합니다.

결과에 동의하지 않으면?

결과 확정 후 이의제기 시스템을 통해 1,000 META를 스테이킹하고 이의를 제출할 수 있습니다. 이의가 인정되면 스테이킹이 반환됩니다.

수수료는 얼마인가요?

패배 풀의 2%만 플랫폼 수수료로 차감됩니다. 베팅 시에는 가스비 외 추가 수수료가 없습니다.

보상은 자동으로 지급되나요?

아니요. 결과 확정 후 '클레임' 버튼을 눌러 직접 보상을 수령해야 합니다. 이는 가스비를 절약하고 사용자 주도권을 보장하기 위함입니다.
