전체 테스트를 실행하고 결과를 리포트합니다.

## test-runner 에이전트 호출

### 전체 테스트 (인자 없음)
1. 단위/통합 테스트 실행:
   - `test:unit` / `test:integration` 스크립트가 있으면 각각 실행 → 개별 보고
   - 스크립트 없으면 `npm test` 결과를 단위+통합 합산으로 보고 (별도 집계 불가)
2. E2E 테스트 (`npx playwright test`)
3. 회귀 테스트 (기존 기능 깨지지 않는지)
4. 결과 리포트 생성 → docs/tests/full-report-{YYYY-MM-DD}.md에 저장

### 범위 지정 (인자 있음)
지정된 범위에 대해 테스트 실행

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 테스트 결과 리포트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[test:unit/test:integration 스크립트 있는 경우]

[스크립트 없는 경우 — 이중 계산 방지]

```
